# 胖青蛙电影
三个tab页，分别是影评、电影、我的。
###功能：
- 电影浏览：
	- 电影分类浏览
	- 电影搜索
	- 电影详情浏览
- 影评浏览：
	- 影评详情浏览
	- 点赞
	- 收藏
	- 评论
- 收藏：
	- 浏览收藏文章
	- 删除收藏
